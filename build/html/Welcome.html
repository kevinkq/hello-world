About reStructured Text¶

This is an example reStructuredText document that I wrote, to get used to working with reStructuredText. reStructuredText is a markdown language that is used to create the CrateDB documentation at Crate. Most parts of the documentation come as stylish and well structured HTML, and it is all done using reStructuredText, which is a WYSISWG plaintext markup syntax and parser system. Using reStructuredText, you can quickly create websites and standalone documentation.

In order to get used to it, I would say it is best to minimize the chatter and get into the writing.
